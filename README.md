# Personal Journal/ Note Taking Web App

A simple, clean, and private personal journal web application built with HTML, Tailwind CSS, and Vanilla JavaScript. This project focuses on core frontend fundamentals while implementing full CRUD (Create, Read, Update, Delete) functionality for managing personal notes.

This app is designed as a beginner-to-intermediate frontend project that demonstrates real-world application logic, state management, and UI design — without using frameworks.

## Features
- Create Notes - Write and save new journal entries
- Read Notes - View all saved notes in a clean layout
- Edit Notes - Update existing notes
- Delete Notes - Remove notes you no longer need
- Responsive UI built with Tailwind CSS
- Fast & Lightweight - No frameworks, no build tools

## Tech Stack
HTML5, Tailwind CSS, JavaScript

### What this project demonstrates
- DOM Manipulation and Event Handling
- State management without frameworks
- CRUD logic implementation
- Clean UI design

## Future Improvements
- Search and filter notes
- Tagging or categories
- Dark mode toggle
- Export notes as PDF or text

## How to Run Locally
1. Clone the repository
```git clone https://github.com/mikeboat45/ ```

2. Navigate to the project folder
```cd note-taking-app```

3. Open ```index.html``` in your browser.
